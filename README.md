# Another Astonishingly Anguished Worker

Live your life to the fullest ! Marry someone, invest, buy a house !

You just have to do that in 20 seconds...

## Requirements

* Python
* Any browser

## Usage

Generate `index.html` and launch it

```bash
$ python3 process.py
$ firefox index.html
```
